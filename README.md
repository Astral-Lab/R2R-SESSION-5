# React Course - Session 5

Welcome to the **React Course** repository for **Session 5**! This repository contains the code and resources for the session.

---

### 📝 Prerequisites
Before you begin, ensure you have the following installed on your system:

* Node.js (LTS version)
* npm (usually comes with Node.js)

If you haven't installed Node.js yet, you can download it from [here](https://nodejs.org/)
.

### 🚀 Getting Started (Locally)

To get started with the React course locally, follow these steps:

1. **Clone the repository**:
   
   ```bash
   git clone https://github.com/Astral-Lab/R2R-SESSION-5.git

2. **Install dependencies**:
   
   ```bash
   npm install

3. **Run the program**:
   
   ```bash
   npm run dev
